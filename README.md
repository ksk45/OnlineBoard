## 概要
ホワイトボードアプリケーション

## 機能一覧

### ログイン（SignIn)ページ
- メールアドレス、パスワードを入力してログイン
- 以下の画面への遷移ボタンを提供：
  - 新規登録（SignUp）ページ
  - ゲストログインページ

### 新規登録（SignUp）ページ
- 名前、メールアドレス、パスワード、確認用パスワードを入力して新規登録
- 以下の画面への遷移ボタンを提供：
  - ログイン（SignIn）ページ

### ゲストログインページ
- 名前を入力してゲストログイン
- 以下の画面への遷移ボタンを提供：
  - ログイン（SignIn）ページ
